# Spotify-Dashboards
## Description
The Spotify Dashboard is an interactive Power BI report designed to analyze music streaming data, artist popularity, album performance, and listening trends. The dashboard helps users explore Spotify streaming insights and identify patterns in music consumption.

The report provides insights into:
- Top streamed songs and artists
- Album and genre performance
- Monthly and yearly listening trends
- Track popularity analysis
- Duration and streaming comparisons
- Interactive filtering for deeper analysis

This dashboard enables users to monitor music trends efficiently and gain data-driven insights into streaming behavior.

---

## Tech Stack
The dashboard was built using the following tools and technologies:

- **Power BI Desktop** – Used for dashboard creation and interactive visualizations
- **Power Query** – Used for data cleaning and transformation
- **DAX (Data Analysis Expressions)** – Used for calculated measures and KPIs
- **Data Modeling** – Used to establish relationships between datasets

### File Formats
- `.pbix` for Power BI project development
- `.png` for dashboard preview images

---

## Data Source
The dataset used in this project contains Spotify streaming data including:
- Track Name
- Artist Name
- Album Name
- Genre
- Streams
- Duration
- Popularity Score
- Release Date

The data was imported into Power BI and transformed using Power Query for analysis and visualization.

---

## Features / Highlights

### Business Problem
Music streaming platforms generate massive amounts of data, making it difficult to identify listening patterns, trending artists, and song performance. A centralized dashboard helps analyze streaming trends and user preferences effectively.

### Goal of the Dashboard
To provide users with an interactive dashboard that helps:
- Analyze music streaming trends
- Identify top-performing songs and artists
- Compare album and genre popularity
- Monitor listening behavior over time
- Support data-driven music insights

---

## Dashboard Highlights
- KPI cards for:
  - Total Streams

- Interactive visualizations including:
  - Top Artists by Streams
